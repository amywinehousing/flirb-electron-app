# flirb-electron-app
*USAGE:*
*AUTOMATIC:*
```
Just clone the source, then cd into the source folder, run chmod +x ./install.sh and then run sudo ./install.sh.
```
*MANUAL:*
**Download the source.**
```
git clone https://github.com/amywinehousing/flirb-electron-app.git
```
**Go into the app directory and run the following commands:**
```
sudo pacman -S nodejs npm
npm install electron
npm start
```
**To use with rofi/dmenu or any other launcher create a .desktop file**
